# Dementia-Diagnosis

## Overview

**Dementia-Diagnosis** is a cognitive assessment application designed to support the screening and evaluation of cognitive impairment through structured tasks, questionnaires, and data-driven analysis. The application is intended for research, educational, and decision-support purposes only and is not a standalone medical diagnostic tool.

The system aims to identify patterns associated with cognitive decline by aggregating performance metrics across multiple cognitive domains such as memory, attention, and executive function. It is designed to support longitudinal tracking and comparative analysis over time.

---

## Features

- Structured cognitive assessment tasks
- Quantitative scoring across multiple cognitive domains
- Consistent, repeatable assessment framework
- Support for longitudinal cognitive tracking
- Modular design for extending assessments and analytics

## Usage

Start the application

python main.py


Complete the cognitive assessment

Users are guided through a series of cognitive tasks and questionnaires.

Responses and performance metrics are recorded automatically during the session.

Review results

The application generates summary scores and domain-level insights.

Results may be displayed within the application or exported for further analysis.

## Intended Use

This application is intended for:

Academic and clinical research

Educational demonstrations

Prototype clinical decision-support tools

Exploratory analysis of cognitive performance data

It is not intended for independent clinical diagnosis or treatment decisions.

## Limitations

Outputs are indicative and probabilistic rather than definitive diagnoses.

Performance may vary across populations, languages, and testing environments.

The application does not replace professional clinical judgment, neuropsychological evaluation, or medical imaging.

## Disclaimer
